# UAMSimulator
K-UAM 연구를 위한 시뮬레이터 개발 프로젝트입니다. 

개발환경: Unity3D, C#

## 정의


1NM(해리) == 1.8532480 km

Transform 1좌표 == 0.001NM

Map의 서울시 지도상 가로 거리 => 72km => 38.8769NM 로 고정함

시간에 대한 정의

1초 => 1분

## 지도

# 원본 지도 이미지 사이즈
7600x4092 (px)
116,923x62,954 (m/반올림)

# 계산식

원본 추출시 축적 비례 : 65px : 1000m

65px : 1000m = 7600px : x
x = 7600*1000/65
x = 116,923.0769230769 (m)

65px : 1000m = 4092px : y
y = 4092*1000/65
y = 62,953.84615384615 (m)
