# UAMSimulator
K-UAM 연구를 위한 시뮬레이터 개발 프로젝트입니다. 

개발환경: Unity3D, C#

## 정의


1NM(해리) == 1.8532480 km

Transform 1좌표 == 0.001NM

Map의 서울시 지도상 가로 거리 => 72km => 38.8769NM 로 고정함

시간에 대한 정의

1초 => 1분

## 지도

지도 중심점 : 복정역

**원본 지도 이미지 사이즈**
*반올림 계산*
```
7600x3784 (px)

138,182x68,800 (m)
```

**계산식**
```
원본 지도 축적 비례 : 110px : 2000m

110px : 2000m = 7600px : x
x = 7600*2000 / 110
x = 138,181.8181818182 (m)

110px : 2000m = 3784px : x
x = 3784*2000 / 110
x = 68,800 (m)
```
